Add an admin

In order to start working with BidiPass you need to add Admins. This could be done by using the command below:

$(npm bin)/bidipass master admin add \
  -k '0x3e722ce009e8acbfad73048108d965b6e38c8d2051d4feaef9fe8d867de7f62c'
